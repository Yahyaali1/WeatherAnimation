# WeatherAnimation

Animatons for weather conditions. Weather animation is a custom view that can incorporated in Android Application. 

#### Conditions Included
- [X] Sunny 
- [X] Cloudy
- [X] Rain 
- [X] Thunderstorm 
- [X] Windy 
- [X] Day/Night
